webpart开发步骤

  1. 在UI层新建一个web窗体

  2. 在UI的webconfig配置数据库（mainDB不用更改）

    Database=demo1;Server=localhost;Uid=sa;Pwd=123123;

  3. 在entiry新建一个类，将数据库中的所有字段设，私有成员变量和属性
    public int account（account对应数据库的字段）
    private int _account = 0;//账号（_account为私有变量，加下划线）

  4. 在DAO里设置数据库查询什么的

  5. 先将system和框架的东西引入

  6. 需继承的地方写上
  7. 命名空间写对
  8. 类前加public，不然默认为private访问不到
  9. 改后台的代码，需重新生成解决方案，必要时清理解决方案
  10. 数据库里必须有个自增的主键 ID int
  11. 逻辑一般需写在BO里，不要用sql语句来判断
  12. 数据库-工具-选项-设计器-表设计器和数据库设计器-阻止保存要求重新创建表的更改
  13. 数据表的字段最好都设上默认值，避免报错
